# Hughes

Unopinionated quasi-monads for Ruby

## Installation

Add this line to your application's Gemfile:

    gem 'hughes'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install hughes

## Usage

Maybe/Nothing:

    try(obj) { |obj| obj.chained.method.calls }
